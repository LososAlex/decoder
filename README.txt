Программа создана Алексеем LososAlex Коноваловым

Программа создана для расшифровки и зашифровки текста состоящего из цифр. Текстовые файлы для зашифровки/расшифровки должны быть в формате txt и находится в папке data

Для корректной работы программы-расшифровки текст должен удовлетворять условиям:
1) Между цифрами(они же зашифрованные буквы) и знаками препинания (кроме тире разделяющего предложение) должен быть один пробел
2) между словами и разделителями предложений(типа тире) должно быть три пробела 

Для корректной работы программы-зашифровки текст не должен содержать цифр.
